# Coding Interview Playlist (Bangla)

This resource contains a curated list of coding interview problems and tutorials created by **Tamim Shahriar**. The videos explain core data structures and algorithms in **Bangla**, making them an excellent resource for Bengali-speaking students preparing for technical interviews.

The problems are organized by topic and sorted roughly from fundamental concepts to more advanced applications.
